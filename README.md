간단한 인벤토리 기능 구현



## 💜 주요기능

- 디자인 패턴
  - Singleton Pattern

- 아이템 장착, 해제기능
- 아이템 장착 시 스텟변경이 스테이터스에 반영
  - 아이템 장착 시 action을 통하여 변경을 status ui에 전달하여 변경 내용을 갱신
